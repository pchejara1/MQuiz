General Instructions

  • Two modes of test module available, i.e. Practice Mode and Exam Mode.
  • "View All Questions" Palette – Displays the status of each question with a colour code, and each colour represents a status that you can view in the View All Questions Cluster.
  • In both modes, if a question has multiple responses, then you will be notified with a prompt message:- Multiple Choice Single Response (MCSR)- Multiple Choice Multiple Response (MCMR).
  • Each correct answer will get 4 marks and
  • Each incorrect answer will deduct 1 mark
  • Skipped questions do not carry any score, but you can reattempt these questions by Filtering them before you complete the test.
  • Each question is allocated with a time of 1 min to answer, which indicates the total time available to complete the test module
  • End Test – If you end the test before completing then it will be considered as an incomplete test

Practice Mode

  • Pause Your Test - You can pause the ongoing practice test anytime by clicking on the "Pause Button" next to the timer, and you can click "Play Button" to resume the practice mode
  • Only in "Practice Mode" questions have the option to "Review Later"
  • In Practice mode, once you attempt a question, you will get the question's solution along with correct and incorrect answer.

Exam Mode

  • In Exam mode, you cannot pause your test.
  • Question Review Later option is not applicable/available in Exam mode
  • No solution will appear in exam mode
